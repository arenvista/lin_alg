# Triangular Matrix

See [[1747598768-NUAP|Laplace Cofactor Expansion]]

$$\begin{bmatrix}
n_1, n_2, n_3\\
0, n_4, n_5\\
0, 0, n_6
\end{bmatrix}$$

A triangular matrix is one such that 0's are found on all entries above | below the main diagonal
